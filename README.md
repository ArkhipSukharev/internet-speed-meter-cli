# Speed Test 

## Запуск 
1. Откройте папку проекта.
2. Дважды кликните `start_speedtest.bat`.
3. Скрипт автоматически:
   - найдет Python;
   - при необходимости установит зависимости;
   - запустит приложение;
   - откроет страницу в браузере (`http://localhost:8501`).

## Параметры в настройках приложения

- `URL файла для скачивания`  
  Адрес файла, по которому выполняются запросы для замера скорости.

- `Количество запросов`  
  Сколько последовательных скачиваний будет выполнено за один тест.  
  Больше запросов = более усредненный результат, но дольше тест.

- `Таймаут запроса (сек)`  
  Максимальное время ожидания одного запроса.  
  Если сервер не отвечает за это время, запрос считается неуспешным.

- `Повторы при 429`  
  Количество повторов, если сервер вернул `429 Too Many Requests`.  
  Нужен для обхода временных ограничений на частые запросы.

- `Пауза между запросами (сек)`  
  Интервал между запросами.  
  Увеличение паузы снижает вероятность `429`, но делает тест дольше.

## Что показывается после теста

- `Скорость (MB/s)` — средняя скорость загрузки.
- `Среднее время` — среднее время одного запроса.
- `Объем данных` — суммарно скачанные данные за тест.
- График и таблица — детали по каждому запросу.
